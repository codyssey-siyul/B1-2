# B1-2 멀티모달 콘텐츠 제작
## 스토리보드  
 **1. 브랜드 아이덴티티**

   + **브랜드명:** Root Coffee
   + **타    겟:** 2-30대(Minimal/Trendy)
   + **톤앤매너:** Green, Cozy, Calm, Sustainable
   + **U  S  P:** 텀블러 사용을 자연스러운 일상으로 만드는 카페
   + **핵심 메시지:** "좋은 습관은 매일의 한 잔에서 시작됩니다."
   + **광고의 목적:** 인지/브랜드가 가진 친환경적 가치와 '텀블러 사용을 일상화하는 카페'라는 차별점을 알린다.<br>'텀블러 할인'을 제시하여 고객이 직접 참여하고 행동으로 옮기도록 이끈다.
